  • Фронтенд для миллионов

    Анастасия Горячева / Авито

    Требовательность аудитории Авито растет постоянно. Мир, технологии – всё меняется. Ожидания пользователей всё выше и выше. Как меняемся мы сами, чтобы дарить больше радости нашим пользователям? Почему эти изменения – внезапно – помогают нам самим развиваться в профессии? Какие боли и выгоды приносит текущий стек технологий, при чем тут дизайн-платформа, есть ли жизнь без react, куда мы катим open source и что будет дальше.

  • Ни RESТа себе! GraphQL is Coming

    Артем Малышев

    Уже очень давно стандартом де-факто для дизайна web API стал REST. Но вот GitHub и Facebook анонсировали поддержку GraphQL API. Зачем они это сделали? Стоит ли нам сделать тоже самое? Какие инструменты для этого предоставляет экосистема Python? Хорошо ли они спроектированы? REST уже всё? Ответы на эти вопросы и не только вы узнаете из моего доклада.
